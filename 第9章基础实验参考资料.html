2．交换机的基本配置  
（
1）给交换机命名。要进入全局模式，才可以给交换机命名。  
switch>enable  
从用户配置模式进入特权配置模式  
从特权配置模式进入全局配置模式  
switch#configure terminal  
switch(config)#hostname fashion  
fashion(config)#  
给交换机命名为fashion  
可以使用no hostname命令来取消命名。

（
2）设置交换机的IP地址及默认网关。使用VLAN1管理本交换机，VLAN1  
接口就是交换机上的管理接口，此接口上的IP地址将用于对此交换机的管理，如  
Telnet、HTTP和SNMP等。  
fashion(config)#interface vlan 1  
进入接口配置模式  
fashion(config-if)#ip address 192.168.10.1 255.255.255.0  
设置交换机的IP地址和子网掩码为192.168.10.1 255.255.255.0  
fashion(config-if)#no shutdown  
启用该接口  
可以在特权配置模式下使用show ip interface或show interface  
vlan1命令来查看交换机的IP地址和管理接口是否开启。  
fashion(config-if)#exit  
回到全局配置模式  
fashion(config)#ip default-gateway 192.168.10.89  
设置交换机默认网关为192.168.10.89，此命令只对S2628G有效。  
使用no ip default-gateway命令可禁止默认网关设置。  
（
3）设置交换机的密码  
fashion(config)#enable password 0 rg  
fashion(config)#line vty 0 4  
设置交换机的特权模式密码为“rg”  
进入线程配置模式  
fashion(config-line)#password 0 star 设置交换机的Telnet登录密码为“star”  
fashion(config-line)#login 启用Telnet的用户名密码验证  
（
4）设置交换机接口速率和通信模式并对接口进行描述  
fashion(config)#int fastethernet 0/1  
fashion(config-if)#speed 100  
fashion(config-if)#exit  
进入1号接口配置模式  
设置1号接口速率为100Mbps  
fashion(config)#line console 0  
fashion(config-line)#speed 9600  
fashion(config-if)#exit  
进入控制口配置模式  
将传输速率设置为9600bps  
fashion(config)#int fa 0/1  
进入1号接口配置模式  
fashion(config-if)#duplex full  
fashion(config-if)# description TO PC1  
设置1号接口为全双工模式  
设置1号接口名字为“TO PC1”  
可以使用no description命令来取消接口命名。  
如果有多个接口要设置可以通过下面的命令来实现：

fashion(config)#int range fa 0/1–5

进入组接口配置模式  
可以使用show interfaces命令来查看接口设置和统计信息。  
（
5）设置系统的日期和时间  
fashion#clock set 16:20:00 12 1 2014 设置系统时钟为2014年12月1日16:20  
fashion#show clock  
显示系统时间信息  
fashion#show version  
显示交换机的系统信息

6）保存设置  
fashion#copy running-config startup-config
